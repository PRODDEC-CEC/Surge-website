# Surge Website

Official website for Surge event.

**Live Site:** [surge-website-iota.vercel.app](https://surge-cec.vercel.app)

---

## Tech Stack

- React.js
- Tailwind CSS
- Create React App

---

## Installation

```bash
git clone https://github.com/PRODDEC-CEC/Surge-website.git
cd Surge-website
npm install
npm start
```

---

## Available Scripts

- `npm start` - Run development server
- `npm run build` - Build for production
- `npm test` - Run tests
